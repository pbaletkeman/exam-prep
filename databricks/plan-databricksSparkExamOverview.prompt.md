# Databricks Certified Associate Developer for Apache Spark
## Exam Overview

### Certification Summary
The **Databricks Certified Associate Developer for Apache Spark** certification validates your understanding of Apache Spark architecture, components, and the ability to apply the Spark DataFrame API to complete practical data manipulation tasks.

---

### Exam Details

| Aspect | Details |

| **Exam Type** | Proctored Certification |

### Exam Coverage by Topic

#### 1. **Apache Spark Architecture and Components - 20%**
- Execution and deployment modes
- Execution hierarchy (Driver, Executors, Tasks)
- Fault tolerance mechanisms
- Lazy evaluation principles
- Garbage collection optimization
- Shuffling operations
- Actions and Transformations
- Broadcasting

#### 2. **Using Spark SQL - 20%**
- Spark SQL functions
- Query execution
- SQL fundamentals with Spark

#### 3. **Developing Apache Spark DataFrame/DataSet API Applications - 30%**
- Selecting and renaming columns
- Manipulating columns
- Filtering rows
- Dropping and sorting rows
- Aggregating rows
- Handling missing/null data
- Combining DataFrames (joins, unions)
- Reading and writing DataFrames
- DataFrame partitioning
- Working with schemas
- UDFs (User Defined Functions)
- PySpark API proficiency

#### 4. **Troubleshooting and Tuning Apache Spark DataFrame API Applications - 10%**
- Performance optimization techniques
- Common failure scenarios
- Debugging approaches
- Tuning Spark applications

#### 5. **Structured Streaming - 10%**
- Streaming data concepts
- Structured Streaming fundamentals
- Real-time data processing

#### 6. **Using Spark Connect to Deploy Applications - 5%**
- Spark Connect basics
- Application deployment

#### 7. **Using Pandas API on Apache Spark - 5%**
- Pandas-compatible syntax on Spark
- DataFrame interoperability
